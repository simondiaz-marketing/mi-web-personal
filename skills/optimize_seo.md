# Skill: SEO Optimization

This skill provides a procedure for auditing and improving the SEO of the website.

## Procedure

1. **Audit Meta Tags**:
   - Check if `<title>` is descriptive and includes relevant keywords (e.g., "Simón Díaz", "Marketing Digital", "NotebookLM").
   - Ensure `<meta name="description">` exists and is between 150-160 characters.
   - Check for Open Graph (`og:`) and Twitter card tags for social sharing.

2. **Heading Hierarchy**:
   - Verify there is exactly one `<h1>` per page.
   - Ensure headings (`<h2>`, `<h3>`, etc.) follow a logical structure without skipping levels.

3. **Semantic HTML**:
   - Use `<main>`, `<header>`, `<footer>`, `<section>`, and `<article>` tags correctly.
   - Ensure `<img>` tags have descriptive `alt` attributes.

4. **Internal Linking**:
   - Check that all internal links are working and use descriptive anchor text.

5. **Keyword Analysis**:
   - Search the page for primary keywords and suggest additions if they are missing from key areas (headers, first paragraph).

## Usage
Run this skill by reviewing `index.html` and other page files against this checklist.
